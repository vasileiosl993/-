Some words about this project

Αυτή είναι μία διαδικτυακή εφαρμογή για την αναγνώριση χειρόγραφων λέξεων. Διαλέγετε την εικόνα που θέλετε να αναγνωρίσετε μέσω του κουμπιού Choose File και στην συνέχεια πατώντας το κουμπί Upload, θα ανακατευθυνθείτε σε μια άλλη σελίδα όπου θα εμφανιστεί η εικόνα σας και το αναγνωρισμένο κείμενο από το μοντέλο μας. Μέσω της μπάρας περιήσηγης μπορείτε να μεταβείτε στην αρχική οθόνη όπου μπορείτε να ακολουθήσετε την ίδια διαδικασία για όσες εικόνες επιθυμείτε. Βεβαιωθείτε ότι ο τύπος εικόνας που θα επιλέξετε είναι .png ή .jpg.
